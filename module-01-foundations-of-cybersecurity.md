# Module 1: Foundations of Cybersecurity

## Goal
Understand what cybersecurity is, why it matters, and how security roles protect people, systems, data, and organizations.

## Key Ideas

- **Cybersecurity:** Protecting systems, networks, devices, and data from unauthorized access, damage, disruption, or theft.
- **CIA Triad:** The three main goals of security.
  - **Confidentiality:** Only authorized people can access information.
  - **Integrity:** Information stays accurate and unchanged unless authorized.
  - **Availability:** Systems and data are accessible when needed.
- **Security posture:** How prepared an organization is to protect, detect, respond, and recover.
- **Threat:** Anything that can cause harm.
- **Vulnerability:** A weakness that can be exploited.
- **Risk:** The chance that a threat exploits a vulnerability and causes impact.

## Security Domains

- Security and risk management
- Asset security
- Security architecture and engineering
- Communication and network security
- Identity and access management
- Security assessment and testing
- Security operations
- Software development security

## Common Cybersecurity Roles

- **SOC Analyst:** Monitors alerts, investigates suspicious activity, and escalates incidents.
- **Security Analyst:** Reviews risks, logs, vulnerabilities, and controls.
- **Incident Responder:** Handles security incidents and recovery steps.
- **GRC Analyst:** Works with governance, risk, compliance, policies, and controls.

## Important Practices

- Use strong passwords and MFA.
- Keep software updated.
- Limit access using least privilege.
- Watch for phishing and social engineering.
- Report suspicious activity quickly.
- Protect sensitive data.

## Quick Review

- Cybersecurity protects confidentiality, integrity, and availability.
- Risk happens when threats and vulnerabilities meet.
- Security is not only tools; it also includes people, processes, policies, and communication.
- Entry-level security work often starts with monitoring, documentation, access review, and basic investigation.
