# Module 7: Automate Cybersecurity Tasks with Python

## Goal
Understand how Python can help automate simple and repeatable cybersecurity tasks.

## Key Ideas

- **Python:** Beginner-friendly programming language often used for automation and data handling.
- **Automation:** Using scripts to perform repeatable tasks faster and more consistently.
- **Variable:** Stores a value.
- **Function:** Reusable block of code.
- **Loop:** Repeats an action.
- **Conditional:** Runs code only when a condition is true.

## Python Basics

- Use variables to store values.
- Use lists to store multiple items.
- Use loops to process repeated data.
- Use conditionals to make decisions.
- Use functions to organize reusable logic.

## Security Automation Examples

- Read log files.
- Count failed login attempts.
- Search for suspicious IP addresses.
- Filter event records.
- Format investigation notes.
- Compare indicators of compromise.

## Simple Analyst Workflow

1. Collect the data.
2. Clean or format the data.
3. Search for important patterns.
4. Summarize the result.
5. Save or report the output.

## Good Scripting Habits

- Keep scripts simple.
- Use clear variable names.
- Add comments only where helpful.
- Test with safe sample data.
- Do not hardcode passwords or secrets.
- Validate input before using it.

## Quick Review

- Python helps reduce repetitive work.
- Automation should be tested and controlled.
- Security scripts are useful for log review, filtering, and reporting.
